# archive-site-dezhoupoker

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面。仓库内容以静态网页文件为主，不针对任何特定域名或网站进行归档。所有页面仅供存档与展示之用。

## 目录结构

```
.
├── index.html          # 入口页面（可选）
├── pages/              # 存放各归档页面
├── assets/             # 静态资源（图片、样式、脚本等）
└── README.md           # 本文件
```

- `pages/` 目录下存放各个独立归档页面，文件名或子目录名可反映页面内容或编号。
- `assets/` 目录用于存放页面所需的公共资源，如图片、CSS、JavaScript 文件等。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接在浏览器中打开查看。
- 页面内容可能包含历史存档信息，仅供学习、研究或参考使用。
- 不保证页面内容的时效性、准确性或完整性。
